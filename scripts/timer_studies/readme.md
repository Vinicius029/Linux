# Temporarizador de Estudo

Script que conometra seu tempo de estudo e salva o progresso em um arquivo

## Como usar

```bash
    # Script possui as seguintes funcionalidades

    # Inicializar
    ./study_timer.sh start

    # Pausar
    ./study_timer.sh pause

    # Verificar status
    ./study_timer.sh status

    # Continuar, após o pause
    ./study_timer.sh continue

    # Finalizar
    ./study_timer.sh finish

    # Histórico
    ./study_timer.sh history
