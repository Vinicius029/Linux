`mkdir` : criar diretórios
```bash
    mkdir teste
```
`mkdir -p` : criar diretórios com outros diretórios dentro
```bash
    mkdir -p teste/teste01
```
`rmdir` : remove o diretórios
```bash
    rmdir teste01
```
`rm -rf` : remove diretórios com outros diretórios dentro
```bash
    rm -rf teste/teste01
```
