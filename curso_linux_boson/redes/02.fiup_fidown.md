# ifdown / ifup
Esse arquivo define como as interfaces de rede devem ser configuradas automaticamente quando o sistema inicia, localizada em `/etc/network/interfaces`

## Estrutura básica do arquivo
Um exemplo típico de configuração:
```bash
    # Interface de loopback
    auto lo
    iface lo inet loopback

    # Interface de rede principal
    auto eth0
    iface eth0 inet dhcp
```
Explicação:
| Palavra-chave            | Função                                                                       |
| ------------------------ | ---------------------------------------------------------------------------- |
| `auto eth0`              | Faz com que a interface `eth0` seja ativada automaticamente na inicialização |
| `iface eth0 inet dhcp`   | Define a interface `eth0` como IPv4 (`inet`) e com IP automático (DHCP)      |
| `iface eth0 inet static` | Define um IP fixo (estático) para a interface                                |

## Exemplo com IP estático
```bash
    auto eth0
    iface eth0 inet static
    address 192.168.1.10
    netmask 255.255.255.0
    gateway 192.168.1.1
    dns-nameservers 8.8.8.8 8.8.4.4
```
toda vez que o sistema iniciar, a interface eth0 receberá sempre o mesmo IP.

## Comandos ifdown e ifup
Esses comandos usam o conteúdo do arquivo `/etc/network/interfaces` para desativar e reativar interfaces de forma controlada.

Desativar uma interface, desliga a interface eth0, liberando o IP e desconectando da rede
```bash
    sudo ifdown eth0
```
Reativa a interface eth0 usando as configurações definidas em /etc/network/interfaces.
```bash
    sudo ifup eth0
```
Reiniciar interface (útil após editar o arquivo)
```bash
    sudo ifdown eth0 && sudo ifup eth0
```

## Resumo 
| Ação                         | Arquivo/Comando              | Função                          |
| ---------------------------- | ---------------------------- | ------------------------------- |
| Configurar rede manualmente  | `/etc/network/interfaces`    | Define se IP é DHCP ou estático |
| Aplicar as alterações        | `ifdown` / `ifup`            | Desliga e religa a interface    |
| Solicitar novo IP automático | `dhclient eth0`              | Requisita IP via DHCP           |
| Verificar o resultado        | `ifconfig` ou `ip addr show` | Mostra IP e status da interface |
